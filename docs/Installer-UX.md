# RoleMesh Installer UX (v0.1)

## 목표
비개발자가 10~15분 내 기본 번들을 구성하도록 한다.

## 플로우
1. 환영 + 환경 점검
2. "내가 가진 것" 체크
   - OpenClaw
   - Claude Code
   - Codex
   - amp
   - Telegram
3. 자동 역할 매핑 제안
   - PM / Builder / Analyst
4. 사용자 확인/수정
5. 연결 검사 (토큰/경로/헬스체크)
6. 워커 시작 + 샘플 요청 실행
7. 완료 리포트

## 질문 예시
- "OpenClaw가 설치되어 있나요? [자동탐지/직접입력]"
- "개발 역할은 Claude Code / Codex 중 어떤 걸 우선할까요?"
- "분석 역할에 amp를 기본으로 사용해도 될까요?"
- "Telegram 알림을 연결할까요?"

## 실패 UX
- 원인 + 해결 버튼형 제안
  - 토큰 누락
  - 실행파일 경로 없음
  - 포트 충돌
- "지금은 라이트 모드로 계속" 옵션 제공

## 라이트 모드
- 최소 구성: PM + Builder 또는 PM + Analyst
- 기능 제한 고지 후 즉시 사용 가능

## 완료 화면
- 구성된 역할표
- 현재 활성 에이전트
- 첫 명령 예시 3개
- 문제 해결 링크 (docs)
